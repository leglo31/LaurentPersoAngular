Bienvenue sur Nouvelles de Rêves, site de publications littéraires

Nouvelles de Rêves

Nouvelles de Rêves est un site crée en collaboration, puis repris à 90% de manière auto-didacte afin de perfectionner mes compétences en javaEE, en SQL pour la base de données, et en HTML5/CSS3 pour la vitrine de ce site avec un soupçon de javaScript. Il est né afin d'offrir aux auteurs une possibilité de présenter leurs écrits devant un public composé à la fois d'auteurs et de simples passionnés de littérature.

Nouvelles de Rêves attend avec impatience d'utiliser tout son potentiel en déployant le code javaEE que j'ai réalisé en début d'année 2021. En effet, contrairement aux langages html/css/javaScript, il est plus difficile de trouver des hébergeurs free pour du code en javaEE. Cependant, ce site "vitrine" vous propose un travail de présentation qui se veut responsive, en utilisant FlexBox et les médias Queries. Les données sont évidement en dur pour le moment, bien qu'elles existent réellement dans une base de données relationnelle. J'ai utilisé de l'open source pour les IDE tels que NetBeans, Bracket, La plateform Java, le serveur Tomcat, le serveur filezilla, l'hébergeur PlanetHoster, phpMyAdmin pour la BDD, Linux dans un premier temps, puis Windows 10 pour terminer.

Vous êtes employeurs, n'hésitez pas à me contacter pour en connaître davantage sur mon parcours, mes compétences et ce projet qui continuera d'évoluer...
